# @commercetools/api-response-normalizer

Helper functions to normalize api responses.

## Install

```bash
npm install --save @commercetools/api-response-normalizer
```

## Supported endpoints

* `/products`
* `/products/:id`
* `/customers`
* `/customers/:id`
* `/product-types`
* `/product-types/:id`
